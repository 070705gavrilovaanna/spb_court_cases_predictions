# Прогнозирование исхода судебных дел в Арбитражном суде СПб

Проект по предсказанию результатов арбитражных дел Санкт-Петербурга на основе данных за 2023 год. Используя мультиклассовую классификацию, построена модель, предсказывающую исход дела с точностью **74%**.

## Этапы работы

**Технологии:**  
`Python`, `Pandas`, `Scikit-learn`, `XGBoost`, `Matplotlib`, `Seaborn`

### 1. Сбор данных
Для сбора данных была проведена работа с официальной Картотекой арбитражных дел (kad.arbitr.ru) - государственным порталом, который содержит полную информацию о судебных процессах. Были отобраны вручную дела Арбитражного суда Санкт-Петербурга и Ленинградской области за 2023 год, просмотрены карточки дел и выгружена ключевая информация (участники, судья, сумма иска, категория и результат) для формирования собственного набора данных.

### 2. Подготовка данных
Работа с датасетом из 367 арбитражных дел 2023 года. Целевая переменная - результат дела:

| Исход | Код |
|-------|-----|
| отказано в принятии иска | 0 |
| истцу отказано | 0 |
| иск удовлетворен частично | 1 |
| иск удовлетворен полностью | 2 |

### 3. Feature Engineering
Создала 18 юридически осмысленных признаков:
- **Типы сторон** - автоматическая классификация истцов/ответчиков (госорганы, частные компании, ИП)
- **Финансовые признаки** - логарифм суммы иска, категории сумм, дела с нулевыми исками
- **Временные признаки** - длительность процесса, месяц подачи, квартал
- **Специфические признаки** - дела против госорганов, банкротства, сложные категории дел

### 4. Моделирование
Сравнила 3 алгоритма:
- RandomForest (**лучший результат**)
- XGBoost
- LogisticRegression

### 5. Результаты

**Лучшая модель:** RandomForest (без дополнительной оптимизации)

| Класс | precision | recall | f1-score |
|-------|-----------|--------|----------|
| 0 (отказ в принятии) | 0.56 | 0.53 | 0.55 |
| 1 (отказ истцу) | 0.67 | 0.50 | 0.57 |
| 2 (удовлетворен) | 0.83 | 0.93 | 0.87 |

- **Accuracy:** 74%
- **Weighted F1:** 0.73
- Лучше всего предсказываются дела с полным удовлетворением иска (F1 = 0.87)
- Важнейший признак: `is_bankruptcy` (дела о банкротстве)

## Оптимизация

- **Балансировка классов** - `class_weight='balanced'` для работы с дисбалансом
- **GridSearchCV** - подбор гиперпараметров для RandomForest с временной кросс-валидацией
- **One-Hot Encoding** - для категориальных признаков
- **Логарифмирование** - суммы исков для нормализации распределения
- **Стратификация** - сохранение распределения классов при split'е

## Что я узнала

- **Мультиклассовая классификация** - задача с 3 классами требует аккуратной интерпретации метрик
- **Ограниченность данных** - 367 дел это небольшой объем, но модель показывает устойчивый результат
- **Интерпретируемость** - анализ важности признаков показал, что дела о банкротстве влияют сильнее всего, также важны тип сторон и сумма иска

## Интерпретация результатов

Модель показала точность **74%** - хороший результат для сложной задачи прогнозирования судебных решений.

- Лучше всего предсказываются дела с полным удовлетворением иска (F1 = 0.87)
- Наибольшую сложность представляют случаи частичного удовлетворения — самый редкий и субъективный исход

Анализ feature importance выявил реальные закономерности: дела о банкротстве, тип сторон и сумма иска статистически значимо влияют на исход.
